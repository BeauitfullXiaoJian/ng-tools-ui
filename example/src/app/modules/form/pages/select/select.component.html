指令参数 [ 所属模块： DropdownModule ]
指令名称 说明
tsDropdown 下拉外部根元素指令，一般使用在一个div元素中，不要给样式
tsToggle 触发器指令，修饰的元素如果被点击将会显示/隐藏面板，一般为按钮，这个元素可以自定义样式
tsDropMenu 下拉面板指令，修饰的元素会隐藏，直到按下触发按钮
属性名称 类型 说明
dropup 不需要设值 添加这个属性将会在触发元素上方显示
组件参数 [ 所属模块： DropdownModule ]

选择器： ts-dropdown

属性名称 类型 说明
dropup 不需要设值 添加这个属性将会在触发元素上方显示
color,sm,lg,outline 查看按钮文档 颜色，尺寸，空心
value any 下拉选中的值，支持双向绑定
items Item[] 下拉选项列表，具体看上方例子：集成下拉
组件参数 [ 所属模块： SelectModule ]

选择器： ts-select

属性名称 类型 说明
dropup 不需要设值 添加这个属性将会在触发元素上方显示
color,sm,lg 同上 颜色，尺寸
value any 下拉选中的值，支持双向绑定
items Item[] 下拉选项列表，注意，如果Item只有value,text两个属性的话，将会显示text内容作为下拉，如果还有content内容那么将会显示content的内容而text会用于查询（value,text是必须要有的）
