搜索组件 @Component

选择器： ts-search

属性名称 类型 说明
color string 颜色设置，bootstrap的基础颜色[danger,light,info...]
placeholder string 提示说明文字
searchLabel string 搜索框提示说明文字
search (key: string) => Observable<Item[]> 查询方法，提供了查询参数，需要返回一个可观察的对象
ngModel Item[] 支持双向绑定，设置、获取当前选中的选项(注意和这个是选项，不是值)
事件名称 事件类型 说明
ngModelChange Item[] 每次选项状态改变时会触发，返回当前所有选中的项目/td>
