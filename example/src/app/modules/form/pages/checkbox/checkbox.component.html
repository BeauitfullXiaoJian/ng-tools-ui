导入选项模块（此模块包含多选，单选，按钮组）

单项选择 @Component

选择器： ts-radio

属性名称 类型 说明
color string 颜色设置，bootstrap的基础颜色[danger,light,info...]
value any 选项的对应的值，当这个选项被选中时可以获取这个值
checked boolean 支持双向绑定，设置、获取当前选项是否被选中
事件名称 事件类型 说明
checkedChange boolean 每次选项状态改变时会触发，返回当前选中状态
groupHandle {{"{ checked: boolean, value?: any }"}} 每次选项状态改变时会触发，返回当前选中状态和选项值，一般用于选项组中的绑定

多项选择 @Component

选择器： ts-checkbox

属性名称 类型 说明
color string 颜色设置，bootstrap的基础颜色[danger,light,info...]
value any 选项的对应的值，当这个选项被选中时可以获取这个值
checked boolean 支持双向绑定，设置、获取当前选项是否被选中
事件名称 事件类型 说明
checkedChange boolean 每次选项状态改变时会触发，返回当前选中状态
groupHandle {{"{ checked: boolean, value?: any }"}} 每次选项状态改变时会触发，返回当前选中状态和选项值，一般用于选项组中的绑定

单项选择群组 @Directive

选择器： *[tsRadioGroup]

属性名称 类型 说明
value any 支持双向绑定，可以获取/设置选项组的值
事件名称 事件类型 说明
valueChange any 每次选项群组中的状态改变时会触发，返回当前选中选项的值

多项选择群组 @Directive

选择器： *[tsCheckboxGroup]

属性名称 类型 说明
values any[] 支持双向绑定，可以获取/设置选项组的值,这是一个所有选中的值的数组
事件名称 事件类型 说明
valueChange any[] 每次选项群组中的状态改变时会触发，返回当前所有选中选项的值

按钮组 @Component

选择器： ts-btns

属性名称 类型 说明
color string 颜色设置，bootstrap的基础颜色[danger,light,info...]
outline 不需要设值 添加这个属性按钮将会使用outline样式
sm 不需要设值 添加这个属性按钮会变小
lg 不需要设值 添加这个属性按钮会变大
multi boolean 是否允许多选，默认false
value any|any[] 当前选中的值，支持双向绑定，如果是单选则是选中的值，多选则是所有选中的值数组
事件名称 事件类型 说明
valueChange any|any[] 每次选项状态改变时会触发，返回当前的选中的值（多选时为数组）
