导入选项模块（此模块包含多选，单选，按钮组）

单项选择 @Component

选择器： ts-radio

属性名称 类型 说明
color string 颜色设置，bootstrap的基础颜色[danger,light,info...]
value any 选项的对应的值，当这个选项被选中时可以获取这个值
disabled boolean 禁用
ngModel any 支持双向绑定，设置、获取当前选中的值
事件名称 事件类型 说明
checkedChange boolean 每次选项状态改变时会触发，返回当前选中状态

多项选择 @Component

选择器： ts-checkbox

属性名称 类型 说明
color string 颜色设置，bootstrap的基础颜色[danger,light,info...]
value any 选项的对应的值，当这个选项被选中时可以获取这个值
disabled boolean 禁用
ngModel any[] 支持双向绑定，设置、获取表单的值（是一个数组）
事件名称 事件类型 说明
checkedChange boolean 每次选项状态改变时会触发，返回当前选中状态
