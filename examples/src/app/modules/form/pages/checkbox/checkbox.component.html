单项选择 @Component

选择器： ts-radio

属性名称 类型 说明
color string 颜色设置，bootstrap的基础颜色[danger,light,info...]
value any 选项的对应的值，当这个选项被选中时可以获取这个值
checked boolean 支持双向绑定，设置、获取当前选项是否被选中
事件名称 事件类型 说明
checkedChange boolean 每次选项状态改变时会触发，返回当前选中状态
groupHandle {{"{ checked: boolean, value?: any }"}} 每次选项状态改变时会触发，返回当前选中状态和选项值，一般用于选项组中的绑定

单项选择群组 @Directive

选择器： *[tsRadioGroup]

属性名称 类型 说明
value any 支持双向绑定，可以获取/设置选项组的值
事件名称 事件类型 说明
valueChange any 每次选项群组中的状态改变时会触发，返回当前选中选项的值

多项选择群组 @Directive

选择器： *[tsCheckboxGroup]

属性名称 类型 说明
values any[] 支持双向绑定，可以获取/设置选项组的值,这是一个所有选中的值的数组
事件名称 事件类型 说明
valueChange any[] 每次选项群组中的状态改变时会触发，返回当前所有选中选项的值
